# Prologue

Welcome to the **Prologue** &mdash; the section that covers everything *around* the code: how to keep your Krayin install up to date and how to contribute back to the project.

If you're new to Krayin, start at the [Introduction](../introduction/README.md) instead &mdash; this section assumes you already have a working install and want to upgrade it or send a patch.

## What's in this section

- [Upgrade Guide](./upgrade-guide.md) &mdash; step-by-step instructions for moving a running Krayin install to the latest release, plus the dependency-change reference.
- [Contribution Guide](./contribution-guide.md) &mdash; how to file bug reports, propose features, pick the right branch, run Pint, and report security vulnerabilities.
